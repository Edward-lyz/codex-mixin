## What's New

- Added Fusion virtual models: Panel models analyze in parallel, Judge compares their results, and Final streams the answer.
- Fusion is no longer limited to the first turn or Plan mode. Later user turns, including coding work, run the full pipeline; tool continuations in the same turn go directly to Final.
- Added “Fusion Settings…” to the macOS menu bar app for configuring Panel, Judge, Final, timeouts, and read-only tools.
- Added workspace-jailed read-only Panel tools and coverage for HTTP, WebSocket, parallel execution, and failure fallback.
- Update prompts now show version-specific changes in the system language.
